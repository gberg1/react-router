React Router Boilerplate Application with Webpack, Babel, and Express

##Steps:
 - `npm install`

###Build for development:
 - `npm run build:dev`

###Navigate to server:
- `nodemon app/server.js` from root directory

###Use Live Reload:
 - `npm start`
 - Navigate to `http://localhost:8080`
 
###Build a minified version for production:
 - `npm run build:prod`
