trekrec

A discovery tool for finding new places and experiences.

##Steps:
 - `npm install`

###Use Live Reload:
 - `npm start`
 - Navigate to `http://localhost:8080`

###Build for development:
 - `npm run build:dev`
 
###Build a minified version for production:
 - `npm run build:prod`

###Navigate to server:
- `nodemon app/server.js` from root directory
